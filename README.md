# V-QRW 
Denerwuje mnie niefektywny transport na mojej trasie. Spóźniam się do pracy, stoję w korkach. Przeklinam zamiast uczyć się lub pracować. Jest tłok, nieprzyjemny odór. Nie mogę sobie odpocząć.
